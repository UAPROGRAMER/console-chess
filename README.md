# Console Chess

Chess made in c++ with ncurses for compatible consoles.

## Libraries

Only ncurses and stdlib.

## Gameplay

You have a cursor you use to move around the board. \
Press 'z' to chose figure under your cursor. \
Press 'z' on another space to try moving selecter figure. \
Press 'x' to canset figure selecton. \
Bottom right square changes color depending on whose turn this is. \
Capture opponent's king to win the game.
